## 🌿 About

**Ubuntu Mind Garden** is a terminal-based mental wellness game written in pure C. It uses evidence-based psychological techniques combined with the rich cultural heritage of Uganda and Africa to help players manage anxiety, depression, stress, and build daily mindfulness habits.

Each session plants a "seed" in your **Mind Garden** — and with consistency, you watch it grow from tiny sprouts → saplings → a mighty Ugandan Mvule tree → a full thriving forest. 🌳

## 🛠 Compile & Run

**Requirements:** GCC, Unix/Linux/macOS terminal (or Windows WSL)

```bash
# Compile
gcc ubuntu_mindgarden.c -o ubuntu_mindgarden -lm

# Run
./ubuntu_mindgarden
```

No external libraries required — pure C standard library + POSIX.

## 🧠 Mental Health Techniques Used

- **4-7-8 Breathing** — Dr. Andrew Weil's anxiety reduction technique
- **5-4-3-2-1 Grounding** — Cognitive grounding for anxiety/panic
- **Positive Affirmations** — CBT-inspired self-talk reframing

- **Progress Gamification** — Habit reinforcement through garden growth
